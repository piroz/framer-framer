# Bun Example

Run framer-framer on Bun.

## Setup

```bash
bun add framer-framer
```

## Run

```bash
bun run index.ts
```

## Notes

- Bun has full npm compatibility — `framer-framer` works out of the box
- All Web standard APIs used by framer-framer are natively available in Bun
- Bun's built-in server uses the `export default { fetch }` pattern
